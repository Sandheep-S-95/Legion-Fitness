# Legion Fitness 💪🏋️‍♀️

## Project Overview 🌟
Full-stack fitness tracking application built with MERN stack and Tailwind CSS, containerized using Docker.

## 🚀 Live Deployed App
### **[LAUNCH LEGION FITNESS 🔥](https://legion-fitness-frontend.vercel.app/)**
*(Transform your fitness journey with one click!)*

## Technologies 💻
- MongoDB
- Express.js
- React
- Node.js
- Tailwind CSS
- Docker
- Vercel Deployment

## Key Features 🔍
- Containerized microservices architecture
- Responsive design
- MVC pattern implementation
- Scalable fitness tracking platform

## Architecture 🏗️
- Separate frontend/backend containers
- Docker Compose orchestration
- Modular application structure

## Deployment 🌐
- Hosted on Vercel
- Containerized with Docker
- Continuous integration workflow
